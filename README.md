# DappDemo
A simple Dapp demo in ethereum network.

## Config
Please check [truffle-config.js](./truffle-config.js).

## Compile
```sh
truffle compile
```

## Deploy
```sh
truffle deploy
```

## Run
```sh
npm run dev
```

NOTE: you should have installed a browser wallet, such as [MetaMask](https://metamask.io/).
